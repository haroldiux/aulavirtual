Repositorio principal del proyecto Aula Virtual LMS UNITEPC.

- Backend: pendiente de implementacion.
- Documentacion base: `PLAN_LMS_UNITEPC.md`, `CONTEXTO_PROYECTO.md`, `AGENTS.md`.

## Repositorios

- Frontend: https://github.com/haroldiux/AULAVIRTUALFE
- Proyecto completo: https://github.com/haroldiux/aulavirtual

## Clonar con frontend

```bash
git clone --recurse-submodules https://github.com/haroldiux/aulavirtual.git
```

Si ya se clono sin submodulos:

```bash
git submodule update --init --recursive
```

## Frontend

```bash
cd Aula-virtual
npm install
npm run dev
```
